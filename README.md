# Adventure-Works-Project-Dashboards

**Project Overview**

Analyzed Adventure Works sales data across time, products, and countries.

Built an interactive Excel dashboard showing sales, profit, and regional performance.

**Data Cleaning**

Removed duplicates, blanks, and formatting issues.

Standardized column names and corrected mismatched IDs.

Ensured clean date formats and proper numeric values.

**Data Modeling**

Designed a Star Schema with Internet Sales as Fact Table.

Dimensions: Date, Product, Sales Territory, Customer.

Connected tables using primary keys and added slicers for filtering.

**Dashboard Highlights**

Interactive visuals with KPIs, charts, and slicers.

Shows year, month, and quarter-wise trends.

Includes product-wise and country-wise performance insights.

**Key Insights**

Clear yearly and monthly sales trends with seasonal peaks.

Top 5 products and monthly product demand patterns.

Strongest quarters and top-performing countries identified.

**Conclusion**

Offers a complete view of business performance.

Helps in forecasting, planning, and data-driven decision-making.
